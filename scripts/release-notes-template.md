## CmdV {VERSION}

A free, open-source clipboard manager for Apple Silicon Macs running macOS 26+, with the native Liquid Glass look.

### Install

1. Download `CmdV-{VERSION}.dmg` below and open it.
2. Drag **CmdV** into **Applications**.
3. First launch: **right-click CmdV → Open → Open**.
   (CmdV is a free open-source app and isn't notarized by Apple, so macOS shows a warning the first time. This is expected.)
4. Press **⇧⌘V** anywhere to open your clipboard history.
5. When asked, grant **Accessibility** permission in System Settings → Privacy & Security → Accessibility so CmdV can paste directly at your cursor. Without it, CmdV still works — it copies the item and you press ⌘V yourself.

> **Note:** after updating CmdV to a newer version you may need to re-grant Accessibility (remove and re-add CmdV in the Accessibility list).

### Highlights

- 🖱️ Menu bar quick copy: click the icon for your pinned + recent items (one click copies); hold or right-click for the options menu; list sizes configurable in Settings
- 📋 Automatic clipboard history — text, links, images, files, colors, code
- 🔍 Instant search (including text inside screenshots, via on-device OCR) and type filters
- 📌 Pinboards for snippets you want to keep forever
- 📚 Paste Stack (⇧⌘C): queue up copies, then ⌘V pastes them back one by one
- ⌨️ Quick Paste ⌘1–⌘9, paste as plain text (⇧Return), multi-select paste
- 🔒 Privacy-first: password managers ignored automatically, per-app ignore rules, pause capture, 100% local — no account, no cloud, no analytics
