# CmdV Codebase Documentation

Architecture reference for contributors and future agent sessions. Companion docs:
[PITFALLS.md](PITFALLS.md) (traps and fixes) and [ROADMAP.md](ROADMAP.md) (future plan).

## Big picture

```
                 ┌─────────────────────────────  every 500 ms  ──────────────────────────┐
                 │                                                                        │
  NSPasteboard ──┤ ClipboardMonitor (@MainActor, changeCount poll)                        │
                 │        │                                                               │
                 │        ▼                                                               │
                 │   PrivacyGate (pure) ── skip: self-copy / concealed types / ignored app│
                 │        │ record                                                        │
                 │        ▼                                                               │
                 │   ClipboardCapture (Sendable snapshot) ──► ClipStore (@ModelActor)     │
                 │                                              │  classify · hash-dedupe │
                 │                                              │  image→disk · thumbnail │
                 │                                              │  prune · async OCR      │
                 │                                              ▼                         │
                 │                                        SwiftData store                 │
                 │                                              │ @Query (main context)   │
                 │                                              ▼                         │
                 │   Shelf UI (non-activating NSPanel + SwiftUI Liquid Glass)             │
                 │        │ Return / double-click / ⌘1-9                                  │
                 │        ▼                                                               │
                 └── PasteEngine (writes pasteboard, flags self-copy) ── CGEvent ⌘V ──► target app
```

Two invariants make the whole design work:

1. **Single writer**: `ClipStore` is the only code that mutates SwiftData. The UI is a pure
   reader via `@Query`. Data crosses actor boundaries only as `Sendable` value types
   (`ClipboardCapture`, UUIDs, plain fields) — never as `@Model` objects.
2. **The shelf never activates the app**. The target app stays frontmost while the panel
   has key focus, which is what makes paste-at-cursor possible at all.

## Module guide

### `App/`
- **CmdVApp.swift** — `@main`. The menu bar item is AppKit-managed (see
  `UI/MenuBar/`), so the only SwiftUI scene is a never-inserted placeholder
  `MenuBarExtra`. Creates `AppState` once with a single `State(initialValue:)`.
- **AppState.swift** — `@MainActor @Observable` root object owning the container, store,
  monitor, shelf controller, and paste-stack controller. `start()` is a no-op under tests
  (`isRunningTests` guard) so unit tests control all state. Every user action funnels
  through here: paste/copy/delete/rename/edit/pin, pinboard CRUD (NSAlert dialogs),
  clear history, Quick Look. Paste optionally snapshots + restores the previous clipboard
  (`Defaults.restoreClipboardAfterPaste`, 0.4 s delay).

### `ClipboardMonitor/`
- **ClipboardMonitor.swift** — polls `NSPasteboard.general.changeCount` on a 500 ms timer
  (macOS has **no** clipboard-change notification API; polling is the Maccy-proven
  approach). Holds `expectedChangeCounts: Set<Int>` — PasteEngine registers the count it
  is about to cause *before* writing, so self-copies are recognized. Reads defensively:
  file URLs via `readObjects`, PNG before TIFF, RTF/HTML only when there is no image,
  promise-only pasteboards skipped. `pause(for:)`/`resume` back the menu-bar pause.
- **PrivacyGate.swift** — pure, fully unit-tested. Ordered verdicts: `skipSelfCopy` →
  `skipTransientType` (8 types incl. `org.nspasteboard.ConcealedType/TransientType/
  AutoGeneratedType`, 1Password, TypeIt4Me, remote-clipboard) → `skipExcludedApp` →
  `record`. The order is contractual — tests pin it.
- **ItemClassifier.swift** — pure. `ClipKind`: text / link / image / file / color / code.
  Colors: hex (#3/4/6/8) and `rgb()`/`rgba()`. Link: single-line web URL with host.
  Code: ≥2 heuristic signals (braces, semicolons, indentation; keywords count double).
- **ClipboardCapture.swift** — the `Sendable` snapshot struct that crosses to `ClipStore`.
- **SourceAppResolver.swift** — `org.nspasteboard.source` if present, else frontmost app;
  icon + display-name helpers.

### `Storage/`
- **Models.swift** — `@Model ClipItem` (unique `uuid`, `contentHash`, kind, text/rtf/html,
  `imageFileName` + `thumbnailData`, file paths, source app, optional `title`, `ocrText`,
  size metadata, optional `pinboard`) and `@Model Pinboard` (name, colorHex, sortOrder,
  inverse `items`).
- **ClipStore.swift** — `@ModelActor`, the single writer. `ingest(_:)`: classify →
  SHA-256 content hash → if an *unpinned* item with the same hash exists, bump its
  `createdAt` instead of inserting → save image to disk + make thumbnail → insert →
  prune → fire-and-forget OCR task that back-fills `ocrText`. Also: delete, clear history
  (unpinned only), rename, edit text, pinboard CRUD, `pinToNewestPinboard`.
  `StoreFactory.makeContainer(inMemory:)` — tests use in-memory.
- **RetentionPolicy.swift** — pure. Age (`maxAgeDays`) and count (`maxItems`) pruning;
  `nil`/≤0 disables either; **pinned items are always exempt**.
- **ImageFileStore.swift** — full images under `App Support/CmdV/Images/` (DB stays
  small); thumbnails via ImageIO `CGImageSourceCreateThumbnailAtIndex` at 600 px max.
  Filenames are validated against path traversal.

### `PasteEngine/`
- **PasteEngine.swift** — `@MainActor`. `copy(item:plainTextOnly:monitor:)` writes the
  richest representations for the kind and calls `monitor.expectSelfCopy()` **first**.
  `copyCapture` (Paste Stack), `copyMultiple` (join with newline), and
  `snapshot()`/`restore()` (`[[String: Data]]`) for clipboard restoration.
- **KeySimulator.swift** — synthetic ⌘V: `CGEvent` keycode 9 + `.maskCommand` posted to
  `.cghidEventTap`. Returns false (copy-only fallback) when Accessibility isn't granted.
- **AccessibilityPermission.swift** — `AXIsProcessTrusted`, prompt (literal
  `"AXTrustedCheckOptionPrompt"` string — see PITFALLS), System Settings deep link.

### `PasteStack/`
- **PasteStackQueue.swift** — pure `Sendable` queue: entries with derived labels,
  `pasteOrder` honoring `isReversed`, `next()` pops from the correct end. Unit-tested.
- **PasteStackController.swift** — `@Observable`. While a stack is active it **claims ⌘V**
  by giving the `.stackPaste` KeyboardShortcuts name a real binding; each trigger pops an
  entry, copies it, *releases the binding*, sends synthetic ⌘V, and re-arms after 0.3 s
  (or deactivates when empty). This avoids both an Input Monitoring permission and a
  self-trigger loop. A stale `.stackPaste` binding is cleared at launch. Owns the small
  glass overlay panel (280×340, top-right, drag-anywhere).

### `UI/Shelf/`
- **ShelfPanel.swift / ShelfPanelController.swift** — the core trick. `NSPanel` with
  `.nonactivatingPanel + .borderless + .resizable`, `canBecomeKey = true`,
  `canBecomeMain = false`, level `.floating`, **`NSApp.activate()` is never called**.
  `show()` records the frontmost app, slides up from the bottom edge (0.22 s).
  `hideAndFocusPreviousApp(then:)` re-activates the target and waits 0.12 s before
  running the paste action so the key window has settled. A local `keyDown` monitor
  routes: Esc (clear search, else hide), ←/→ selection, Return / ⇧Return paste,
  Space Quick Look, Delete, ⌘1–9 quick paste, ⌘C/F/R/E/N/⇧N/A. Global click monitor +
  `windowDidResignKey` dismiss. Width is pinned on resize; height persists.
- **ShelfViewModel.swift** — selection, search text, kind/pinboard filters, rename/edit
  state, visible-item bookkeeping for ⌘1–9 badges.
- **ShelfRootView.swift** — `@Query` newest-first; filter pipeline pinboard → kind →
  search (`ClipItem.matches(query:)` covers title, text, **ocrText**, source app, file
  names). Header = search field + kind chips + pinboard chips. Card strip = `LazyHStack`
  in a `ScrollViewReader`, one `.glassEffect(.regular, in: .rect(cornerRadius: 26))`.
- **ClipCardView.swift** — 230 pt cards, selection ring, ⌘n badge, click/⌘-click/⇧-click
  selection, double-click paste, `.draggable` (file URL / PNG / text via Transferable),
  full context menu.
- **CardPreviews.swift** — per-kind preview views; links fetch favicon/title through the
  `LinkMetadataCache` in `.task(id:)`.
- **QuickLookPreview.swift** — `QLPreviewPanel` data source (`@preconcurrency`); text
  items preview via temp files.

### `UI/StackOverlay/`, `UI/Settings/`, `UI/MenuBar/`
- **PasteStackOverlayView.swift** — `GlassEffectContainer` with `glassEffectID` morph
  animations (the *only* place morphs are used — perf), numbered paste order, reverse
  toggle, per-entry remove.
- **Settings** — TabView (480 pt): General (launch at login via `SMAppService.mainApp`,
  retention pickers, menu bar quick-list counts, restore-clipboard toggle), Privacy
  (Accessibility status refreshed on `didBecomeActive`, ignored-apps list via NSOpenPanel),
  Shortcuts (recorders + cheat sheet). Hosted by **SettingsWindowController** (plain
  NSWindow) because with no SwiftUI menu left there is no place for `SettingsLink`,
  and the `Settings` scene cannot be opened from AppKit.
- **UI/MenuBar/** — **StatusItemController** owns an `NSStatusItem` (not MenuBarExtra —
  SwiftUI cannot tell gestures apart). A plain click pops the *quick copy list*:
  pinned + recent items (counts configurable in Settings, defaults 5/10), each with a
  kind icon or image thumbnail; clicking one copies it. Press-and-hold (≥0.35 s,
  tracked synchronously via `NSApp.nextEvent` in `.eventTracking` mode) or right-click
  opens the *options menu* (open shelf, Paste Stack, pause, clear history, settings,
  quit). Menus pop via the attach-menu-then-`performClick` pattern and detach in
  `menuDidClose`. **QuickMenuQuery** (testable fetch split) and **QuickMenuLabel**
  (pure one-line label derivation) back the quick list.

### `Hotkeys/`, `Support/`
- **HotkeyDefinitions.swift** — `.openShelf` (⇧⌘V), `.pasteStack` (⇧⌘C), `.stackPaste`
  (intentionally no default — bound only while a stack is active).
- **Defaults.swift** — UserDefaults keys + registered defaults (retention 30 d / 1000
  items, poll 0.5 s, shelf height 320) and the `isRunningTests` check.
- **ScriptingHooks.swift** — Darwin-notification hooks for scripting/headless testing:
  `com.tonmoybishwas.CmdV.toggleShelf` and `.togglePasteStack` (see CLAUDE.md).
- **OCRService.swift** — Vision `VNRecognizeTextRequest` (accurate, off-main).
- **LinkMetadata.swift** — `@MainActor` LPMetadataProvider cache.
- **ColorParsing.swift** — `Color(hexString:)`.

| What | Where |
|---|---|
| SwiftData store | `~/Library/Application Support/CmdV/CmdV.store` |

## Testing

Swift Testing (`@Test` / `#expect`), hosted inside the app (`TEST_HOST` in project.yml —
the app skips engine startup under tests via `isRunningTests`). 38 tests:

- `PrivacyGateTests` — all 8 ignored types (parameterized), verdict ordering, app excludes
- `ItemClassifierTests` — color/link/code/prose tables
- `RetentionPolicyTests` — age, cap, pinned exemption, combined
- `PasteStackTests` — order, reverse, remove, labels, clear
- `QuickMenuLabelTests` / `QuickMenuQueryTests` — menu bar quick list labels and the
  pinned/recent fetch split (real in-memory SwiftData store)
- `SmokeTests` — test target sanity

Deliberately untested: UI, panel behavior, CGEvent posting (require a GUI session; verify
those via the headless hooks in CLAUDE.md plus the user's hands).

## Build & release pipeline

- **project.yml** (XcodeGen) → `make gen` → committed `CmdV.xcodeproj`. Bundle
  `com.tonmoybishwas.CmdV`, `LSUIElement` (no Dock icon), hardened runtime, **not
  sandboxed** (sandbox forbids CGEvent posting), strict concurrency `complete`.
- **CI** (`.github/workflows/ci.yml`) — macos-26 runner, Xcode 26.2: xcodegen → build →
  test on every push/PR. Proves fresh-clone reproducibility.
- **Release** (`scripts/release.sh <version>`) — Release build (unsigned) → codesign with
  `$CMDV_SIGN_IDENTITY` → "CmdV Signing" → ad-hoc fallback chain → DMG via
  `scripts/make-dmg.sh` (create-dmg with hdiutil fallback) → force-tag `v<version>` →
  `gh release create` with notes templated from `scripts/release-notes-template.md`.
  A commented `notarytool` block marks where Developer ID notarization slots in.
- **scripts/dev-sign-setup.sh** — guides creating the persistent self-signed "CmdV
  Signing" cert (GUI-only; see PITFALLS on TCC resets).
